О функциях:

Функции в JavaScript - это одна из ключевых концепций, которые позволяют вам определить блок кода, который можно вызывать многократно для выполнения определенной задачи. Функции в JavaScript можно рассматривать как подпрограммы или маленькие программы внутри вашей основной программы. Вот основные аспекты работы функций в JavaScript:

  1. Определение функции:

    Функцию можно определить с помощью ключевого слова function. Ниже пример определения функции, которая складывает два числа:

    function add(a, b){

    return a + b;

    }

    В данном примере add - это имя функции, а a и b - это параметры (или аргументы) функции.

  2. Вызов функции:

    Для выполнения кода, определенного в функции, вы можете вызвать функцию, указав ее имя и передав значения для параметров. Вот пример вызова функции add:

    var result = add(2, 3);

    console.log(result); // Выведет 5

    В этом примере функция add вызывается с аргументами 2 и 3, и результат (сумма) сохраняется в переменной result.

  3. Возвращение значения:

    Функция может возвращать значение с помощью ключевого слова return. В предыдущем примере функция add возвращает сумму a + b.

  4. Локальные переменные:

    Переменные, определенные внутри функции с помощью var, let или const, являются локальными для этой функции и не видимы за ее пределами.

  5. Глобальные переменные:

    Переменные, объявленные вне функций, называются глобальными и доступны во всех функциях. Однако их использование должно быть осторожным, чтобы избежать конфликтов и неожиданного поведения.

  6. Функции как объекты:

    В JavaScript функции также являются объектами. Вы можете передавать функции как аргументы в другие функции, возвращать их из функций и хранить в переменных.

  7. Анонимные функции:

    JavaScript также поддерживает анонимные функции, которые не имеют имени. Они часто используются в качестве аргументов для других функций, как функции обратного вызова (callback functions).

    var multiply = function(a, b) {

    return a * b;

    };

    console.log(multiply(4, 5)); // Выведет 20
